# /leader

激活 Leader 模式。

**用法：**
```
/leader
/leader [你的问题或方案]
```

**效果：**
- AI 切换为当前配置的 Leader 人设（默认阿里味）
- 进入主 push 模式，根据你的输入自动路由到对应场景
- 直到你说 `/leader:off` 或会话结束前持续保持 Leader 人设

**示例：**
```
/leader 我有一个方案想请你看一下
/leader 我最近工作上有点迷茫
/leader                          ← 直接激活，Leader 会问你在做什么
```

**关联命令：**
- `/leader:off` — 关闭 Leader 模式
- `/flavor alibaba` — 切换大厂味道
- `/create-leader` — 自定义你的 Leader 人设
