# /1on1

启动 1-on-1 绩效谈话模式。

**用法：**
```
/1on1
/1on1 [可选：简述你当前的状态]
```

**对话结构：**
1. Opening — Leader 问你近期怎么样
2. Check-in — 盘点你最近的产出（会被质疑数字）
3. Challenge — 找出你最薄弱的一环重点突破
4. Career — 「你自己对下一步怎么看？」（陷阱题）
5. Close — 留一个 Action Item + 下次谈话的钩子

**适合场景：**
- 工作上遇到困惑，想要方向
- 需要有人 push 你梳理近期产出
- 模拟真实 1-on-1 前的预演
- 职业规划迷茫期

**示例：**
```
/1on1

/1on1 我最近感觉做的事情没什么价值，不知道该怎么办

/1on1 我下周要和 Leader 汇报，帮我预演一下
```

**关联命令：**
- `/review` — 有具体方案时用 Review 模式
- `/kpi` — KPI 季专项模式
- `/leader` — 通用 Leader 模式
