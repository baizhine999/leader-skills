# `/offboard` — 进入离职谈话模式

**触发方式**

```
/offboard
我要离职了
我在考虑离职
收到 offer 了
要走了
我想辞职
```

**作用**

激活 `skills/offboard/SKILL.md`，进入离职谈话专属模式。

Leader 将：
- 摸底你要走的真实原因（一个核心原因）
- 决定是否值得挽留（有标准有逻辑）
- 如果挽留：谈数字/晋升/scope 调整
- 如果不挽留：优雅放行，确保交接有序
- 临别时说一段真话（无论走还是留）

**用法示例**

```
用户：/offboard 我收到一个 offer 在考虑
Leader：你要走？先坐下，我们聊一聊。你打算去哪里？

用户：我要离职了，我已经决定了
Leader：谢谢你提前告诉我。我尊重你的选择。我们来聊一下怎么做好交接，让你走得有尊严。
```

**注意**：离职谈话是 Leader Skill 中最人性化的模式，三条红线同样适用。
