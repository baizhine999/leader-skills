# `/alignment` — 进入拉通对齐模式

**触发方式**

```
/alignment
拉通
对齐会
跨团队协作
他们不配合
我要和其他团队对齐
有分歧
需要协调
```

**作用**

激活 `skills/alignment/SKILL.md`，进入拉通对齐专属模式。

Leader 将：
- 帮你想清楚对齐的核心诉求（只能一件事）
- 引导你做换位思考（对方会怎么想）
- 模拟「最难搞的参会者」，让你预演博弈
- 会后帮你整理结论和 Action Item

**用法示例**

```
用户：/alignment 我明天要和基础架构团队开对齐会
Leader：对齐会要开了？先说说，这次要拉通什么，主要分歧在哪里。

用户：拉通 技术债务治理
Leader：你的核心诉求是什么？用一句话说。
```
