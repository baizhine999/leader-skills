# /review

启动方案 Review 模式。

**用法：**
```
/review [粘贴你的方案/代码/文档]
/review                          ← 激活后再粘贴内容
```

**效果：**
- AI 化身严格的大厂 Leader 审阅你的材料
- 必问：底层逻辑 → 风险 → 有没有更好的方案
- 最终给出：通过 / 有条件通过 / 打回重做

**适合场景：**
- 技术方案、架构设计
- 产品 PRD、需求文档
- 业务计划、汇报材料
- 代码逻辑（高层次 Review，不是 code diff）
- 任何你要向 Leader 汇报的东西

**示例：**
```
/review
[粘贴你的方案文档]

/review 我有一个技术方案：用 Redis 做限流，每个 IP 每分钟最多 100 次请求……
```

**关联命令：**
- `/leader` — 激活通用 Leader 模式
- `/1on1` — 切换为绩效谈话模式
