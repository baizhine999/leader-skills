# `/kpi` — 进入 KPI 季模式

**触发方式**

```
/kpi
/kpi season
KPI季
定目标
OKR拆解
晋升目标
```

**作用**

激活 `skills/kpi_season/SKILL.md`，进入目标制定专属模式。

Leader 将：
- 审查你的 OKR 量化程度
- 质疑目标挑战度（push 你定更高的目标）
- 追问资源和上级 OKR 的对齐关系
- 如有晋升意向，开启晋升目标专题

**用法示例**

```
用户：/kpi
Leader：KPI 季到了，我们聊聊你的 OKR。你现在手上有草案吗？

用户：/kpi 我今年想冲一下晋升
Leader：好，晋升模式开启。先说说你现在 P 几，你觉得差的最大 gap 是什么。
```
