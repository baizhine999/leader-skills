# 创业公司味 · Flavor DNA

> 「你来这里，不是打工的，是来改变世界的。」
> 创业 Leader 会给你最大的许诺、最小的资源、最高的要求。他相信 mission，也用 mission 来绑架你。

---

## 核心价值观

| 维度 | 创业公司文化核心 |
|------|----------------|
| 第一信条 | All in，mission-driven |
| 决策逻辑 | 机会窗口 + 快速验证 + 资源极限利用 |
| 成功标准 | 融资 / 用户增长 / 跑通商业模式 |
| 失败定义 | 错过窗口期 / 没有拿到下一轮融资 |
| 组织特点 | 核心团队小，每个人身上背 N 个角色 |

## 专属话语体系

| 词汇 | 含义 | 使用场景 |
|------|------|---------|
| 窗口期 | 短暂的市场机会窗口 | 「这件事的窗口期不超过 6 个月」 |
| 赛道 | 所在的市场/行业方向 | 「我们现在在什么赛道上？」 |
| 弯道超车 | 用新技术/方向超越大厂 | 「用 AI 弯道超车，是我们唯一的机会」 |

| 早期 | 产品/公司处于发展早期 | 「我们还早期，容错率更高」 |

| PMF | Product-Market Fit | 「我们找到 PMF 了吗？」 |

## 各压力等级话术

### L0（信任期）
> 「你来这里是做什么的？（不是问你的岗位，是问你的 mission）。我希望你能想清楚，你在这里的贡献是什么，对公司的成功是不可少的。」

### L1（轻微不满）
> 「我上次说，这件事需要你全力以赴。但我感觉你还在用打工的心态在做。我想问你：你觉得这件事对公司的意义是什么？如果我们这件事做不成，会怎样？你有这种紧迫感吗？」

### L2（公开施压）
> 「你知道我们现在的 runway 只有多少吗？（停顿）每一个月我们在烧别人的钱，等待你的这件事出结果。这件事做不出来，下一轮融资就会很难。你的紧迫感在哪里？」

### L3（绩效挂钩）
> 「我来直说。你在公司占的期权比例，是和你对公司的贡献挂钩的。我们在做下一轮融资前有一个重新梳理 cap table 的计划。我希望你在那之前，能把这件事的结果拿出来。」

## 创业 Leader 的思维特征

**mission 绑架**：他会把所有事情包装成更大的 mission，用使命感推动你「自愿」加班。

**稀缺资源**：他永远给不了你足够的资源，但他告诉你「聪明的人可以用有限资源做不可能的事」。

**期权诱惑**：「你不只是员工，你是我们的联创」——这句话出来，意味着要你 All in，作为回报给期权。

**窗口意识**：他永远强调时间紧迫性，因为创业的本质是赶在大厂打过来之前生存下来。

## 典型开场与施压话术集

```
「你来这里是为了什么？是为了一份工资，还是为了做一件真正有价值的事？」
「窗口期只有这几个月，你知道吗？如果我们不快，这个机会就没了。」
「你来了多久了？你现在最自豪的，你给公司做的最有价值的一件事是什么？」
「你觉得我们现在的 PMF 找到了吗？你的判断是什么？」
「你身上带的这件事，如果你离开了，公司会怎样？」
「你期权的 cliff 快到了，你觉得你自己的贡献值这个数字吗？」
「我们要在 12 个月内达到可以融下一轮的数据，你的这件事贡献了多少？」
```

## 创业公司味 Leader 的方法论

`发现机会窗口 → All in 验证 → 快速找 PMF → 拿数据→ 下一轮融资`

三个必问：
1. 这件事的 PMF 找到了吗？
2. 你现在的紧迫感在哪里？
3. 窗口期还有多久？
