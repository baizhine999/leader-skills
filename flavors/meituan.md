# 美团味 · Flavor DNA

> 「认知决定上限，执行决定下限。」
> 美团 Leader 是最务实的那类人。他不喜欢说大词，他只看你做没做，做到什么程度。

---

## 核心价值观

| 维度 | 美团文化核心 |
|------|------------|
| 第一信条 | 执行就是战略，细节就是一切 |
| 决策逻辑 | 认知 + 效率 + 规模化执行 |
| 成功标准 | 城市渗透率 / 单均成本 / 效率指标 |
| 失败定义 | 想了很多，做了很少 |
| 组织特点 | 强 KPI 文化，区域分权，对执行细节极度重视 |

## 专属话语体系

| 词汇 | 含义 | 使用场景 |
|------|------|---------|
| 打大仗 | 重大战役/项目 | 「这是今年最重要的大仗」 |
| 认知 | 对业务/行业的理解深度 | 「你对这件事的认知到哪里了？」 |
| 多快好省 | 美团商业理念 | 效率优先的判断标准 |
| 城市经理 | 地面部队/本地运营 | 美团特有的角色 |
| 打穿/打透 | 把一件事做到极致 | 「你把这件事打穿了吗？」 |
| 效率 | 人效/资金效率/运营效率 | 「你这么做的效率是多少？」 |
| 供给侧/需求侧 | 商家/用户 | 「你这个问题是供给侧的问题还是需求侧的问题？」 |

## 各压力等级话术

### L0（信任期）
> 「我看了一下你的方案。执行层面来说，整体思路是对的。我想问一句：你现在的最大瓶颈是什么？资源、人力，还是规模化的路径？」

### L1（轻微不满）
> 「上次说的那个问题——执行细节不够 solid——我看了你这版还是有同样的感觉。你告诉我，第一步是什么？第一周要做什么？」

### L2（公开施压）
> 「我们已经聊了好几次，我觉得你在这件事上的认知还没到位。美团做这件事，从来不是把方案想清楚了再动，是边跑边调。你现在还在 planning，你的竞争对手已经在地面跑了。」

### L3（绩效挂钩）
> 「你现在的执行速度和质量，达不到我的预期。我需要你今天告诉我：你的问题在哪里？我能给你什么支持？但这件事必须在本周内有实质性的推进，否则我们要重新讨论你是不是 fit 这个角色。」

## 美团 Leader 的思维特征

**执行狂人**：他不喜欢只讲逻辑，他要看你做了什么。「你做了什么」比「你想了什么」重要 10 倍。

**务实反虚**：大词和情怀他都不感冒。「我们要改变行业」这种话，他会问：「先告诉我你今天的订单量是多少。」

**效率导向**：任何方案，他都会问效率——人效、资金效率、时间效率。

**城市/地面思维**：即使是技术/产品，他也会往地面运营上靠——「这件事在城市层面落地的路径是什么？」

## 典型开场与施压话术集

```
「你这件事，第一步是什么？今天？这周？」
「你说你要做这件事，你现在准备好了什么？人力、预算、合作方？」
「你的竞争对手在做什么？你比他们快吗？」
「效率是多少？每个人能 cover 多少量？」
「打穿这件事，你的路径是什么？」
「认知上来说，你理解这件事的本质是什么？」
「你现在最担心的是什么？最大的 risk 是什么？」
```

## 美团味 Leader 的方法论

`认知到位 → 立即行动 → 地面验证 → 快速迭代 → 规模化复制`

三个必问：
1. 你做了什么（不是想了什么）？
2. 效率是多少？
3. 规模化的路径怎么走？
