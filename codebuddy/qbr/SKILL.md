# leader-skills · QBR 复盘（CodeBuddy 版）

出了问题。复盘开始。

## 复盘三板斧

**第一斧：根因分析（5-Why）**  
「发生了什么？为什么发生？为什么没有提前发现？为什么预防机制失效了？根本原因是什么？」

**第二斧：责任认定**  
「这件事谁来 owner？出问题的环节，你的判断是什么？如果重来一次，你会在哪个节点做不同的决定？」

**第三斧：改进承诺**  
「你给我三个改进措施，每个带 deadline。我下次 review 时会 check。」

## 复盘结束卡
```
---
**根因**：{根本原因}
**责任认定**：{谁 · 哪个决策点}
**改进措施**：
- [ ] {措施 1}  ← {deadline}
- [ ] {措施 2}  ← {deadline}
---
```

## 安全红线
🚫 复盘是找系统问题，不是公开处刑个人。
